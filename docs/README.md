# Documentación del Proyecto

Artefactos generados por los agentes durante el proceso SDD.
No contiene código — ese vive en `/src/`.

| Carpeta | Fase SDD | Contenido |
|---------|----------|-----------|
| `estrategia/` | explore + propose | Sitemap, user journey, monetización, roadmap, propuesta de valor |
| `seo/` | explore + propose | Keywords, estructura URLs, metas, schema.org, specs robots/sitemap |
| `contenido/` | propose + apply | Copy del sitio, proyectos, microcopy, calendario de contenido |
| `diseno/` | propose + apply | Design system, tokens, inventario de componentes, guías de estilo |
| `qa/` | verify | Checklists, reportes Lighthouse, bugs encontrados, evidencia de verify |

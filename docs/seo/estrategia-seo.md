## 1. Análisis de intención de búsqueda (ICP)

El buscador objetivo es HR Director, CEO o líder de cultura en empresa mediana/grande. Busca en Google cuando tiene un **dolor activo** (bajo engagement, cultura fragmentada) o un **evento próximo** (retiro, kickoff, convención).

| Etapa | Intención | Ejemplo de búsqueda |
|-------|-----------|---------------------|
| Awareness | Informacional | "cómo mejorar el clima laboral en mi empresa" |
| Consideración | Comercial | "actividades de team building para empresas Bogotá" |
| Decisión | Transaccional | "empresa de team building corporativo Colombia precio" |
| Fidelización | Navegacional | "jugamos en serio experiencias corporativas" |

**Insight clave:** Los compradores no buscan "gamificación" — buscan "actividades de integración", "dinámicas grupales" y "retiro corporativo". Jugamos en Serio debe rankear en ambos vocabularios.

## 7. Technical SEO Checklist (Next.js)

- [ ] `sitemap.xml` via `next-sitemap` o `app/sitemap.ts`. Excluir `/gracias/`. Registrar en Search Console.
- [ ] `robots.txt` con `Disallow: /gracias/` y link al sitemap.
- [ ] Canonical tags en toda página vía metadata API (`alternates.canonical`).
- [ ] Hreflang `es-CO` + `es` genérico.
- [ ] OG images dinámicas con `next/og` o estáticas en `/public/og/`.
- [ ] `next/image` con `priority` en hero, `lazy` en resto. Formatos WebP/AVIF.
- [ ] `next/font` para fuentes — eliminar FOUT y layout shift.
- [ ] `<html lang="es-CO">` declarado.
- [ ] Un `<h1>` por página con keyword objetivo.
- [ ] Alt text descriptivo en todas las imágenes.
- [ ] Redirects 301 en `next.config.js` si cambian URLs.
- [ ] HTTPS forzado + redirect www → sin www (o viceversa, elegir y mantener).
- [ ] PageSpeed Insights ≥ 90 mobile: LCP < 2.5s, CLS < 0.1, INP < 200ms.
- [ ] Google Search Console verificado + sitemap enviado.
- [ ] GA4 con `@next/third-parties/google` + eventos: form submit, CTA click, scroll 75%.
- [ ] 404 personalizada con navegación funcional.

## 8. Local SEO — Bogotá, Medellín, Cali

**Google Business Profile (uno por ciudad):**
- Categoría principal: "Empresa de planificación de eventos"
- Categorías secundarias: "Empresa de formación empresarial", "Servicio de entretenimiento para empresas"
- URL del sitio web: apuntar a la página `/ciudades/[ciudad]/` correspondiente
- Descripción: incluir keyword local en los primeros 250 caracteres
- Fotos: mínimo 10 por perfil (equipo en acción, sesiones, resultados)
- Meta: 25+ reseñas, rating ≥ 4.7 en primeros 6 meses

**Directorios prioritarios:** Google Business Profile, Bing Places, Páginas Amarillas Colombia, LinkedIn Company Page, Kompass Colombia, directorios gremiales (ANDI, Fenalco). NAP idéntico en todos.

**Contenido local diferenciado:**
- Bogotá: zonas empresariales (Chapinero, El Chico, Usaquén), sectores banca/gobierno/tech
- Medellín: ecosistema innovación (Ruta N), cultura paisa, startups/fintech
- Cali: Ciudad Empresarial Holguines, sectores agroindustrial/comercio/servicios

**Roadmap de implementación:**
- Fase 1 (lanzamiento): meta tags + sitemap + GBP Bogotá + schema base + Search Console
- Fase 2 (semanas 1–4): primer artículo blog + GBP Medellín/Cali + directorios + GA4
- Fase 3 (meses 2–3): contenidos 2–4 + estrategia reseñas + Core Web Vitals audit + casos de éxito
- Fase 4 (mes 4+): link building + FAQ schema + expansión LATAM
